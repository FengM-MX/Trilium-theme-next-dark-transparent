# 主题Trilium-theme-next-dark-transparent使用
- 新增笔记
- 文件类型选择Css
- 新增属性
  ```
  #appTheme=dark-transparent #appThemeBase=next-dark
  ```
- 激活的主题，右上角可以看到主题开关
