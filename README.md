# 主题Trilium-theme-next-dark-transparent使用
- 新增笔记
- 文件类型选择Css
- 新增属性
  ```
  #appTheme=dark-transparent #appThemeBase=next-dark
  ```
- 激活的主题，右上角可以看到主题开关
# 使用样式transparent-appCss
- 直接使用样式，也可以达到主题一样的效果，此时要在设置中，选择主题深色

- 文件类型选择Css
- 新增属性

# 快捷键控制
- 增加快捷键
  - ctrl+[：减少缩进
  - ctrl+]：增加缩进
  - alt+1,2,3,4,5,6,0：分别设置2,3,4,5,6级以及正文
  - 增加点击文档，直接在新的tab页面打开，并切换至新的tab页面
- 使用
  - 新建代码文档，类型选择js frontend
  - ctrl+r刷新，即可
